## Power BI Financial Performance Dashboard
### Project Overview

This project showcases an interactive Power BI dashboard built to analyze financial performance using a sample financial dataset. The dashboard focuses on units sold, profit, discounts, manufacturing price, and segment-wise performance, presented through clear visuals and professional design.

The goal of this project is to demonstrate Power BI skills such as data preprocessing, DAX calculations, visualization design, and interactive reporting.

 ### Data Import & Preprocessing

Imported dataset into Power BI Desktop

Performed data cleaning using Power Query

Fixed:

Column data types (Decimal, Fixed Decimal, Date)

Discount Band datatype issues

Blank and error values

Created proper date hierarchy (Year, Month)

### DAX Measures Created
